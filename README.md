# 🛍️ Raflimited — Интернет-магазин мужской одежды

Добро пожаловать в официальный репозиторий интернет-магазина **Raflimited** — стиль, качество и уверенность в каждой детали.  
Наш проект представляет собой современный, адаптивный и удобный онлайн-магазин мужской одежды, разработанный с акцентом на пользовательский опыт и производительность.

---

## 🚀 Основные возможности

- 👕 Каталог товаров с фильтрацией по категориям, размерам и цене
- 🔍 Поиск по названию и описанию товара
- 🛒 Корзина покупок с возможностью редактирования и оформления заказа
- ❤️ Избранное — сохраняйте понравившиеся товары
- 📦 Страница товара с детальным описанием, фото и размерной сеткой
- 📧 Форма обратной связи и подписка на рассылку новинок
